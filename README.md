Banking System API – Mini Project:

A simple, secure, and modular Banking System built using REST APIs to perform essential banking operations like Deposit, Withdraw, Balance Check, and Money Transfer.
This project showcases practical backend skills, API development, error handling, and clean architecture—perfect for interview discussions and portfolio presentation.

Features:

 1. Create Account:

Auto-generated account number

Stores user details (name, created date, status)

Initial balance starts at 0


 2. Deposit Money:

Adds money to the user account

Validates account and amount

Updates account balance with transaction record


3. Withdraw Money:

Ensures sufficient balance

Throws exception for invalid or high withdrawal requests

Records every transaction


4. Check Balance:

Fetches current balance

Validates account existence


5. Transfer Money:

Transfer funds from one account to another

Atomic operations using transaction management

Detailed error handling
